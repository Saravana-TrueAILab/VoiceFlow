# VoiceFlow

![VoiceFlow Banner](https://via.placeholder.com/1200x500.png?text=VoiceFlow)

VoiceFlow is an AI-powered meeting assistant designed to streamline your workflow by automatically transcribing meetings, generating smart summaries, and extracting actionable tasks. It integrates seamlessly with popular video conferencing tools like Zoom, Microsoft Teams, and Google Meet to ensure you never miss a detail.

-   **Real-time Transcription**: Get 99% accurate speech-to-text in over 30 languages with speaker diarization.
-   **Smart Summaries**: AI-generated bullet points, key takeaways, and sentimental analysis to quickly digest long meetings.
-   **Action Item Extraction**: Automatically identifies commitments and tasks from conversations and assigns them due dates.
-   **Seamless Integration**: Connects with your favorite tools like Slack, Notion, Jira, Salesforce, and HubSpot.
-   **Analytics Dashboard**: Gain insights into meeting productivity, sentiment trends, and team participation.
-   **Kanban Task Management**: Visualize and manage extracted tasks directly within the application.
-   **Secure & Private**: Enterprise-grade security ensures your meeting data remains confidential.

## ğŸ Getting Started

Follow these steps to get the project up and running on your local machine.

### Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn
-   Git

### Installation

1.  **Clone the repository**
    Open your terminal and run the following command to clone the project to your local machine:
    ```bash
    git clone https://github.com/Saravana-TrueAILab/VoiceFlow.git
    ```

2.  **Navigate to the project directory**
    ```bash
    cd VoiceFlow
    ```

3.  **Install dependencies**
    Run the following command to install all necessary packages:
    ```bash
    npm install
    # or if you use yarn:
    # yarn install
    ```

4.  **Start the development server**
    Once the installation is extensive, start the app locally:
    ```bash
    npm run dev
    # or
    yarn dev
    ```

5.  **Open the app**
    Navigate to `http://localhost:5173` in your browser to view the application.
